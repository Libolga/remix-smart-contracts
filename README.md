# Remix IDE Smart Contracts Project

Проект содержит смарт-контракты и тесты для изучения Solidity.

## Контракты
- Storage.sol - простой контракт для хранения значения
- Owner.sol - управление владельцем контракта  
- Ballot.sol - система голосования
- ERC20.sol - токен стандарта ERC-20

## Тестирование
- JavaScript тесты с Hardhat
- Solidity тесты в Remix IDE
